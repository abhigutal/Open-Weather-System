1] Extract "Weather_System.zip" in D drive.

2] Go to the "weather_system/Project/Openweather" folder and open command prompt from address bar.

3] Then type "pip install -r requirements.txt" this command in cmd and install all packages.

4] Next type "python -c "import secrets; print(secrets.token_urlsafe(48))" in cmd , it provide a unique secrete key copy it and replace it with the secrete_key into the "weather.env" file.

5] Install MongoDB compass and make a connection in it, if you already have then dont make a new connection.

6] At last go to app.py file and open it with python application or VS code and run it. It will automatically open into your default browser.



NOTE: Your PC must be connected to stable internet connection. This project requires internet to run this on localhost:5000 port.
